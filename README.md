# Advent Of Code 2024

## Testing

### Test everything

```bash
cargo test --workspace
```

### Test a crate or lib

```bash
cargo test -p day06
```

or

```bash
cargo test -p day14
```

###

## Run solution

```bash
cargo run -- -d 1 -p 1 #Run solution for day 1 part 1
```
or 

```bash
cargo run -- --day 13 --part 2 #Run solution for day 13 part 2
```
